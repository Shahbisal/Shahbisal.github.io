aaaaaAAAAAAA!!! Remastered - Project Details

Genre: Extreme Sports Simulator, Arcade

Role: Environment Artist, Gameplay Programmer (Physics and UI)

Overview

This is a full remaster of a beloved arcade skydiving sequel, modernized for current-gen consoles and PC. My contributions focused on optimizing the core free-fall physics simulation for smooth 60fps performance and revamping the entire user interface to be highly responsive and customizable across different platforms.

Technical Contributions:

  • **Custom Physics:** Fine-tuned the drag and terminal velocity calculations for realistic free-fall behavior.
  • **Responsive UI:** Developed a new UI/UX system that scales perfectly from 4K monitors down to mobile resolutions.
  • **Optimized Assets:** Reworked legacy environment assets to meet modern optimization standards while preserving the original artistic vision.
